## MODIFIED Requirements

### Requirement: カバー画像を削除できる
システムは設定済みのカバー画像を削除できなければならない（SHALL）。削除ボタンをクリックした際、確認ダイアログを表示し、ユーザーが承認した場合のみ削除を実行しなければならない（SHALL）。

#### Scenario: 作品のカバー画像を削除する
- **WHEN** ユーザーが作品詳細ページでカバー画像の削除ボタンをクリックし、確認ダイアログで承認する
- **THEN** システムは画像ファイルを削除し、作品タイルをプレースホルダー表示に戻す

#### Scenario: 出演者のカバー画像を削除する
- **WHEN** ユーザーが出演者詳細ページでカバー画像の削除ボタンをクリックし、確認ダイアログで承認する
- **THEN** システムは画像ファイルを削除し、出演者タイルをプレースホルダー表示に戻す

#### Scenario: 確認ダイアログでキャンセルする
- **WHEN** ユーザーがカバー画像の削除ボタンをクリックし、確認ダイアログでキャンセルする
- **THEN** システムはカバー画像を削除せず、現在の表示を維持する
