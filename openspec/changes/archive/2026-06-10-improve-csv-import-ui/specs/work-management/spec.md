## MODIFIED Requirements

### Requirement: 作品のCSVインポート
システムはCSVファイルから複数の作品を一括登録できなければならない。CSVの列は `title`（作品名）、`performer_names`（出演者名、カンマ区切り）、`performer_furiganas`（ふりがな、カンマ区切り・任意）、`directory_path`（ディレクトリパス・任意）の4列に限定する。この機能は作品ページ内の「一括登録」ボタンから起動するモーダルUIで提供される（サイドバー独立ページではない）。インポート操作は「プレビュー → 確認 → 結果」の3ステップで構成される。

#### Scenario: 作品ページから一括登録を開始する
- **WHEN** ユーザーが作品ページの「一括登録」ボタンをクリックする
- **THEN** CSVアップロードUIを含むモーダルが開く

#### Scenario: 正常なCSVをインポートできる
- **WHEN** ユーザーが正しい形式のCSVファイルをアップロードする
- **THEN** システムはプレビュー（行ごとのバリデーション結果）を表示する

#### Scenario: プレビューで valid 行がデフォルト全選択される
- **WHEN** プレビューが表示される
- **THEN** `is_valid === true` の全行がチェック済み状態になり、`is_valid === false` の行はチェック不可（disabled）で表示される

#### Scenario: プレビューで行ごとにインポート対象を切り替えられる
- **WHEN** ユーザーが valid 行のチェックボックスをクリックする
- **THEN** その行のインポート選択状態がトグルされる

#### Scenario: 全選択・全解除が機能する
- **WHEN** ユーザーが「全選択」ボタンをクリックする
- **THEN** `is_valid === true` の全行が選択状態になる
- **WHEN** ユーザーが「全解除」ボタンをクリックする
- **THEN** `is_valid === true` の全行が未選択状態になる

#### Scenario: 1件以上選択時に確認画面へ進める
- **WHEN** ユーザーが1件以上を選択した状態で「確認へ →」ボタンをクリックする
- **THEN** 選択済み行のみを表示した確認画面に遷移する

#### Scenario: 確認画面にエラー行は表示されない
- **WHEN** 確認画面が表示される
- **THEN** 選択済みの valid 行のみが表示され、エラー行は含まれない

#### Scenario: 確認画面でも行をインポート対象から除外できる
- **WHEN** ユーザーが確認画面で行のチェックボックスを外す
- **THEN** その行はグレーアウトで表示されたまま残り、インポート対象から除外される（再チェックで復帰可能）

#### Scenario: 確認画面からプレビューに戻れる
- **WHEN** ユーザーが確認画面の「← 戻る」ボタンをクリックする
- **THEN** プレビュー画面に戻り、選択状態が維持される

#### Scenario: 確認後にインポートが完了する
- **WHEN** ユーザーが確認画面で「インポート実行」を押す
- **THEN** システムは選択済み行の作品を保存し、出演者を紐づける

#### Scenario: performer_furiganas が省略可能
- **WHEN** CSVに `performer_furiganas` 列が存在しない、または値が空のとき
- **THEN** システムはふりがななしで出演者を登録し、エラーにしない

#### Scenario: title が空の行はエラーとして表示する
- **WHEN** CSVに `title` が空の行が含まれている
- **THEN** システムはその行をエラーとしてプレビューに表示し、その行はインポートしない

#### Scenario: 既存の出演者名と一致した場合は既存出演者を使用する
- **WHEN** CSVの `performer_names` に既存の出演者と同名のエントリが含まれる
- **THEN** システムは新規作成せず、既存の出演者を作品に紐づける

#### Scenario: プレビューのテーブルレイアウトが読みやすい
- **WHEN** プレビューが表示される
- **THEN** テーブルの列は「チェックボックス・タイトル・出演者」の3列で構成される
- **AND** エラー行のチェックボックスセルには XCircle アイコンが表示され、マウスオーバーでエラー内容がツールチップとして表示される
