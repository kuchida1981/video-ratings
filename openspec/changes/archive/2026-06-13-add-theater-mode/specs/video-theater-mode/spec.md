## ADDED Requirements

### Requirement: インライン動画からシアターモードへの切り替え
インライン再生中の動画にホバーすると、右上にシアターモード切り替えボタンが表示される SHALL。ボタンをクリックするとシアターモードに移行する。

#### Scenario: ホバーでシアターボタンが表示される
- **WHEN** インライン再生中の動画要素にカーソルを乗せる
- **THEN** 動画右上にシアターモードボタンが表示される

#### Scenario: シアターボタンのクリックでシアターモードに移行する
- **WHEN** シアターモードボタンをクリックする
- **THEN** ビューポート全体を覆う黒背景のオーバーレイが表示され、動画がビューポート全幅で再生される

---

### Requirement: シアターモードの表示
シアターモードは `position: fixed` でビューポート全体を覆うオーバーレイとして表示される SHALL。動画はビューポート全幅を使い、アスペクト比を維持して表示される。ネイティブのビデオコントロール（再生・シーク・音量・全画面ボタンを含む）が使用できる。

#### Scenario: シアターモードでビューポート全体が覆われる
- **WHEN** シアターモードが有効な状態
- **THEN** サイドバーを含むページ全体がオーバーレイで覆われ、動画がビューポート全幅で表示される

#### Scenario: シアターモード中にネイティブコントロールが使用できる
- **WHEN** シアターモードで動画が再生されている
- **THEN** ブラウザネイティブのビデオコントロール（再生/一時停止・シーク・音量・全画面）が操作できる

#### Scenario: シアターモードから OS 全画面に移行できる
- **WHEN** シアターモード中にネイティブコントロールの全画面ボタンをクリックする
- **THEN** OS レベルの全画面表示に移行する

#### Scenario: OS 全画面からシアターモードに戻る
- **WHEN** OS 全画面表示中に Esc キーを押す
- **THEN** シアターモードのオーバーレイに戻る（動画の再生は継続する）

---

### Requirement: シアターモードの終了
シアターモードは ✕ ボタンで閉じることができる SHALL。

#### Scenario: ✕ ボタンでシアターモードを閉じる
- **WHEN** シアターモード中に ✕ ボタンをクリックする
- **THEN** オーバーレイが閉じ、通常のページ表示に戻る
