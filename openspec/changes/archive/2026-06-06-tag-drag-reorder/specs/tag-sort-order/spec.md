## ADDED Requirements

### Requirement: タグカテゴリをドラッグで並び替えられる
タグ管理UIにおいて、ユーザーはタグカテゴリをドラッグ操作で並び替えできる。並び替えは entity_type（作品用／出演者用）のセクション内のみで有効であり、セクションをまたいだ移動はできない。

#### Scenario: カテゴリをドラッグで並び替える
- **WHEN** タグ管理UIでカテゴリのドラッグハンドルをドラッグして別の位置にドロップする
- **THEN** カテゴリの並び順が変更され、UIに即座に反映される

#### Scenario: 並び替えた順序がサーバーに保存される
- **WHEN** カテゴリのドラッグ操作が完了する
- **THEN** 新しい並び順が API 経由でサーバーに保存される

#### Scenario: entity_type をまたいだドラッグはできない
- **WHEN** 「作品用カテゴリ」セクション内のカテゴリを「出演者用カテゴリ」セクションにドラッグしようとする
- **THEN** ドラッグ操作は無効化され、元の位置に戻る

---

### Requirement: カテゴリ内のタグをドラッグで並び替えられる
タグ管理UIにおいて、ユーザーはカテゴリ内のタグをドラッグ操作で並び替えできる。並び替えは同一カテゴリ内のみで有効であり、カテゴリをまたいだ移動はできない。

#### Scenario: タグをドラッグで並び替える
- **WHEN** タグ管理UIでタグのドラッグハンドルをドラッグして同一カテゴリ内の別の位置にドロップする
- **THEN** タグの並び順が変更され、UIに即座に反映される

#### Scenario: 並び替えたタグの順序がサーバーに保存される
- **WHEN** タグのドラッグ操作が完了する
- **THEN** 新しい並び順が API 経由でサーバーに保存される

---

### Requirement: タグ管理UIの並び順が作品・出演者詳細ページに反映される
タグ管理UIで設定した並び順は、作品詳細ページおよび出演者詳細ページのタグ付けUIに反映されなければならない。

#### Scenario: 作品詳細ページのカテゴリ表示順がタグ管理の並び順に従う
- **WHEN** ユーザーが作品詳細ページのタグ付けセクションを表示する
- **THEN** タグカテゴリがタグ管理UIで設定した `sort_order` 昇順で表示される

#### Scenario: 作品詳細ページのタグ表示順がタグ管理の並び順に従う
- **WHEN** ユーザーが作品詳細ページのタグ付けセクションを表示する
- **THEN** 各カテゴリ内のタグがタグ管理UIで設定した `sort_order` 昇順で表示される

#### Scenario: 出演者詳細ページのカテゴリ表示順がタグ管理の並び順に従う
- **WHEN** ユーザーが出演者詳細ページのタグ付けセクションを表示する
- **THEN** タグカテゴリがタグ管理UIで設定した `sort_order` 昇順で表示される

---

### Requirement: 新規作成されたカテゴリは末尾に追加される
新しいタグカテゴリを作成したとき、そのカテゴリは既存カテゴリの末尾（最大 sort_order + 1）に追加されなければならない。

#### Scenario: 新規カテゴリが末尾に表示される
- **WHEN** ユーザーが新しいタグカテゴリを作成する
- **THEN** 新しいカテゴリは同一 entity_type の既存カテゴリの末尾に表示される

---

### Requirement: 新規作成されたタグは末尾に追加される
新しいタグをカテゴリに追加したとき、そのタグはカテゴリ内の既存タグの末尾（最大 sort_order + 1）に追加されなければならない。

#### Scenario: 新規タグが末尾に表示される
- **WHEN** ユーザーが既存カテゴリに新しいタグを追加する
- **THEN** 新しいタグはそのカテゴリの既存タグの末尾に表示される
