## MODIFIED Requirements

### Requirement: タグの管理
システムはカテゴリに属するタグを作成・編集・削除できなければならない。タグはスコア（整数、nullable）を持つ。スコアがない場合はラベル専用タグとして機能する。タグの名前とスコアは作成後も変更できなければならない。

#### Scenario: スコアありのタグを作成できる
- **WHEN** ユーザーがスコアに整数値を入力してタグを作成する
- **THEN** システムはスコア付きのタグを保存する

#### Scenario: スコアなしのタグを作成できる
- **WHEN** ユーザーがスコアを入力せずにタグを作成する
- **THEN** システムはスコア null のタグを保存する（ラベル専用）

#### Scenario: タグ名を編集できる
- **WHEN** ユーザーが既存のタグの名前を変更して保存する
- **THEN** システムは更新された名前でタグを保存する

#### Scenario: タグのスコアを編集できる
- **WHEN** ユーザーが既存のタグのスコアを変更して保存する
- **THEN** システムは新しいスコアでタグを保存し、そのタグが付いた作品・出演者のスコアが再計算される

#### Scenario: タグのスコアを削除できる（スコアなしにできる）
- **WHEN** ユーザーが既存のタグのスコアを空にして保存する
- **THEN** システムはスコアを null に更新し、そのタグはスコア計算に影響しなくなる

#### Scenario: タグの編集をキャンセルできる
- **WHEN** ユーザーがタグ編集フォームでキャンセルする
- **THEN** システムはタグを変更せず、編集フォームを閉じる

#### Scenario: タグを削除すると既存のタグ付けも削除される
- **WHEN** ユーザーがタグを削除する
- **THEN** システムはそのタグを作品・出演者から外し、スコア計算に影響しなくなる
