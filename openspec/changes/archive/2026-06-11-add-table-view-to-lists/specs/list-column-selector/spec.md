## ADDED Requirements

### Requirement: 作品テーブルの表示列をフィルタパネル内で選択できる
システムは作品テーブル表示時、フィルタパネル内にバッジ形式の列選択UIを表示しなければならない（SHALL）。列選択UIはテーブル表示時のみ表示し、タイル表示時には表示してはならない（SHALL NOT）。

選択可能列: メーカー、シリーズ、スコア、タグ、ファイル数、登録日、カスタム項目（定義済み全件）

#### Scenario: テーブル表示時に列選択UIが表示される
- **WHEN** ユーザーが作品一覧をテーブル表示に切り替える
- **THEN** フィルタパネル内に列選択バッジが表示される

#### Scenario: タイル表示時に列選択UIが表示されない
- **WHEN** ユーザーが作品一覧をタイル表示で見ている
- **THEN** フィルタパネルに列選択バッジは表示されない

#### Scenario: 列バッジをクリックして表示列を切り替える
- **WHEN** ユーザーがメーカーの列バッジをクリックする
- **THEN** 選択状態が反転し、テーブルのメーカー列の表示/非表示が切り替わる

---

### Requirement: 作品テーブルの列選択は localStorage に永続化する
ユーザーが選択した作品テーブルの表示列は、`video-ratings:works-table-columns` キーで localStorage に保存しなければならない（SHALL）。保存値は選択中の列キーの文字列配列とする。デフォルト表示列はメーカーとスコアとする。

#### Scenario: 列選択がブラウザリロード後も保持される
- **WHEN** ユーザーが列選択を変更した後にページを再読み込みする
- **THEN** テーブル表示に切り替えると変更した列設定が復元される

#### Scenario: デフォルト列が適用される（初回起動時）
- **WHEN** localStorage に作品テーブルの列設定が保存されていない状態でテーブル表示に切り替える
- **THEN** メーカー列とスコア列が選択された状態で表示される

---

### Requirement: 出演者テーブルの表示列をフィルタパネル内で選択できる
システムは出演者テーブル表示時、フィルタパネル内にバッジ形式の列選択UIを表示しなければならない（SHALL）。テーブル表示時のみ表示する。

選択可能列: 作品数、作品平均スコア、合計スコア、タグ、カスタム項目（定義済み全件）

#### Scenario: テーブル表示時に列選択UIが表示される
- **WHEN** ユーザーが出演者一覧をテーブル表示に切り替える
- **THEN** フィルタパネル内に列選択バッジが表示される

#### Scenario: 列バッジをクリックして表示列を切り替える
- **WHEN** ユーザーが作品数の列バッジをクリックする
- **THEN** 選択状態が反転し、テーブルの作品数列の表示/非表示が切り替わる

---

### Requirement: 出演者テーブルの列選択は localStorage に永続化する
ユーザーが選択した出演者テーブルの表示列は、`video-ratings:performers-table-columns` キーで localStorage に保存しなければならない（SHALL）。デフォルト表示列は作品数と作品平均スコアとする。

#### Scenario: 列選択がブラウザリロード後も保持される
- **WHEN** ユーザーが出演者テーブルの列選択を変更した後にページを再読み込みする
- **THEN** テーブル表示に切り替えると変更した列設定が復元される

#### Scenario: デフォルト列が適用される（初回起動時）
- **WHEN** localStorage に出演者テーブルの列設定が保存されていない状態でテーブル表示に切り替える
- **THEN** 作品数列と作品平均スコア列が選択された状態で表示される

---

### Requirement: カスタム項目は列選択UIに動的に追加される
システムはカスタム項目定義を API から取得し、テーブルの選択可能列に動的に追加しなければならない（SHALL）。カスタム項目が追加・削除されると列選択UIも自動的に更新される。

#### Scenario: カスタム項目が列選択UIに表示される
- **WHEN** ユーザーが作品一覧のテーブル表示を開き、カスタム項目「品番」が定義されている
- **THEN** 列選択UIに「品番」のバッジが表示される

#### Scenario: 削除されたカスタム項目は列選択UIから消える
- **WHEN** カスタム項目「品番」が削除された後にページをリロードする
- **THEN** 列選択UIに「品番」のバッジは表示されない
