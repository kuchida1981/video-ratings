## REMOVED Requirements

### Requirement: 作品テーブルのソート可能列ヘッダーでソートを切り替えられる
**Reason**: ソート操作はフィルタパネルのソートボタンに一本化する。ヘッダークリックによるソートはフィルタパネルのUIと重複しており、操作経路を統一して UX をシンプルにする。
**Migration**: 作品のソートはフィルタパネルのソートボタン（スコア順・登録日順・タイトル順・出演者順・カスタム項目）で行う。

### Requirement: 出演者テーブルのソート可能列ヘッダーでソートを切り替えられる
**Reason**: 同上。フィルタパネルのソートボタンに統一する。
**Migration**: 出演者のソートはフィルタパネルのソートボタン（名前順・作品数順・作品平均点数順・合計スコア順・カスタム項目）で行う。
