## ADDED Requirements

### Requirement: リスト表示における新規タブ遷移のサポート
作品一覧および出演者一覧の各アイテム（タイルおよびテーブル内のリンク要素）は、`react-router-dom` の `Link` コンポーネントを使用し、HTML の `<a>` 要素としてレンダリングされなければならない (SHALL)。これにより、ブラウザの標準的な「新しいタブで開く」操作（中クリック、修飾キーを押しながらのクリック）が機能しなければならない (SHALL)。

#### Scenario: タイル表示での新規タブ遷移
- **WHEN** 作品一覧または出演者一覧のタイルを「Shift+クリック」または「中クリック」したとき
- **THEN** ブラウザの標準挙動により、該当する詳細ページが新しいタブで開かれる。

#### Scenario: テーブル表示での新規タブ遷移
- **WHEN** 作品一覧または出演者一覧のテーブル内のタイトル部分を「Shift+クリック」または「中クリック」したとき
- **THEN** ブラウザの標準挙動により、該当する詳細ページが新しいタブで開かれる。

#### Scenario: 通常の左クリックによる遷移
- **WHEN** 作品一覧または出演者一覧のタイル、またはテーブル内のタイトル部分を（修飾キーなしで）左クリックしたとき
- **THEN** `react-router-dom` の標準的なSPA遷移により、現在のタブ内で高速に詳細ページへ遷移する。
