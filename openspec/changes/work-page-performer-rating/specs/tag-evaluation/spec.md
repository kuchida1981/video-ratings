## MODIFIED Requirements

### Requirement: 出演者へのタグ付け
システムは出演者に対して出演者用タグを付けられなければならない。出演者評価はグローバルで、全作品のスコア計算に影響する。タグの付け外しは出演者詳細ページおよび作品詳細ページの両方から行える。

#### Scenario: 出演者にタグを付けられる
- **WHEN** ユーザーが出演者詳細画面または作品詳細画面で出演者用タグを選択する
- **THEN** システムはそのタグを出演者に紐づけ、その出演者が主演の全作品の合計スコアが更新される

#### Scenario: 出演者からタグを外せる
- **WHEN** ユーザーが出演者詳細画面または作品詳細画面で出演者に付いたタグを外す
- **THEN** システムはタグの紐づけを削除し、関連する作品の合計スコアが更新される
