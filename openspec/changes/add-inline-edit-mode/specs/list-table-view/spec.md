## MODIFIED Requirements

### Requirement: 作品テーブルの行クリックで詳細ページへ遷移する
作品テーブルの各行をクリックするとその作品の詳細ページへ遷移しなければならない（SHALL）。ただし編集モードON時は、行全体のクリックによるページ遷移は無効化される。タイトル列のリンクは編集モード中も維持される。

#### Scenario: テーブル行クリックで作品詳細へ
- **WHEN** ユーザーが編集モードOFF時に作品テーブルの行をクリックする
- **THEN** その作品の詳細ページへ遷移する

#### Scenario: 編集モード中は行クリックで遷移しない
- **WHEN** ユーザーが編集モードON時に作品テーブルのカスタムフィールドセルをクリックする
- **THEN** 詳細ページへの遷移は発生せず、セルの入力欄にフォーカスが移る

#### Scenario: 編集モード中もタイトルリンクで遷移できる
- **WHEN** ユーザーが編集モードON時にタイトル列のリンクをクリックする
- **THEN** その作品の詳細ページへ遷移する
