# Spec: google-search-links

## Purpose

出演者・作品の詳細ページおよび一覧テーブル（編集モード時）に Google 検索リンクを表示する機能。出演者名や作品タイトルをキーワードとして Google 検索を素早く実行できる。

## Requirements

### Requirement: 出演者ページのGoogle検索リンク
出演者詳細ページのヘッダーに、出演者名と `is_search_keyword = true` の出演者用テキスト型カスタム項目の値（値が空でない場合のみ）を組み合わせた Google 検索リンクを表示しなければならない（SHALL）。キーワードはすべてダブルクォートで囲む。リンクは新しいタブで開く。編集モード中は非表示にする。

#### Scenario: 通常表示時にリンクが表示される
- **WHEN** 出演者詳細ページを表示している（編集モードでない）
- **THEN** ヘッダー付近に Google 検索アイコンリンクが表示される

#### Scenario: カスタム項目なし時の検索クエリが正しく生成される
- **WHEN** 出演者名が「山田太郎」で is_search_keyword = true の項目が存在しない場合
- **THEN** リンク先 URL のクエリパラメータが `"山田太郎"` をエンコードした値になっている

#### Scenario: is_search_keyword = true のカスタム項目値が検索クエリに追加される
- **WHEN** 出演者名が「山田太郎」で is_search_keyword = true の text 型カスタム項目「事務所」の値が「ABCプロダクション」である
- **THEN** リンク先 URL のクエリパラメータが `"山田太郎" "ABCプロダクション"` をエンコードした値になっている

#### Scenario: カスタム項目値が空の場合はクエリに含めない
- **WHEN** 出演者名が「山田太郎」で is_search_keyword = true のカスタム項目の値が空文字である
- **THEN** リンク先 URL のクエリパラメータが `"山田太郎"` のみをエンコードした値になっている

#### Scenario: 編集モード中はリンクが非表示になる
- **WHEN** 出演者詳細ページの編集モードを有効にしている
- **THEN** Google 検索リンクが表示されない

### Requirement: 作品ページのGoogle検索リンク
作品詳細ページのヘッダーに、全出演者名・作品タイトル・`is_search_keyword = true` の作品用テキスト型カスタム項目の値（値が空でない場合のみ）を組み合わせた Google 検索リンクを表示しなければならない（SHALL）。各キーワードはダブルクォートで囲む。リンクは新しいタブで開く。編集モード中は非表示にする。

#### Scenario: 通常表示時にリンクが表示される
- **WHEN** 作品詳細ページを表示している（編集モードでない）
- **THEN** ヘッダー付近に Google 検索アイコンリンクが表示される

#### Scenario: カスタム項目なし時の出演者複数の場合のクエリが正しく生成される
- **WHEN** 出演者が「山田太郎」「鈴木花子」で作品タイトルが「ローマの休日」で is_search_keyword = true の項目がない場合
- **THEN** リンク先 URL のクエリが `"山田太郎" "鈴木花子" "ローマの休日"` をエンコードした値になっている

#### Scenario: 出演者なしの場合のクエリが正しく生成される
- **WHEN** 出演者が登録されておらず作品タイトルが「ローマの休日」の場合
- **THEN** リンク先 URL のクエリが `"ローマの休日"` をエンコードした値になっている

#### Scenario: is_search_keyword = true のカスタム項目値が検索クエリに追加される
- **WHEN** 作品タイトルが「ローマの休日」で is_search_keyword = true の text 型カスタム項目「品番」の値が「MIDE-123」である
- **THEN** リンク先 URL のクエリが `"performer1" ... "ローマの休日" "MIDE-123"` をエンコードした値になっている

#### Scenario: カスタム項目値が空の場合はクエリに含めない
- **WHEN** 作品タイトルが「ローマの休日」で is_search_keyword = true のカスタム項目の値が空文字である
- **THEN** リンク先 URL のクエリには空文字のカスタム項目値が含まれない

#### Scenario: 編集モード中はリンクが非表示になる
- **WHEN** 作品詳細ページの編集モードを有効にしている
- **THEN** Google 検索リンクが表示されない

### Requirement: 出演者一覧テーブルの編集モード時にGoogle検索リンクを表示する
出演者一覧テーブルにおいて、編集モードが有効なとき、各行の出演者名の横に Google 検索アイコンリンクを表示しなければならない（SHALL）。検索クエリは出演者名と `is_search_keyword = true` の出演者用テキスト型カスタム項目の値（値が空でない場合のみ）をダブルクォートで囲んだ形式とする。リンクは新しいタブで開く。編集モードが無効のときは表示しない。

#### Scenario: 編集モード時に検索アイコンが表示される
- **WHEN** 出演者一覧テーブルの編集モードが有効である
- **THEN** 各行の出演者名の横に Google 検索アイコンリンクが表示される

#### Scenario: 通常表示時に検索アイコンが表示されない
- **WHEN** 出演者一覧テーブルの編集モードが無効である
- **THEN** 各行の出演者名の横に Google 検索アイコンリンクは表示されない

#### Scenario: カスタム項目値を含む検索クエリが生成される
- **WHEN** 出演者名が「山田太郎」で is_search_keyword = true の text 型カスタム項目「事務所」の値が「ABCプロダクション」のとき検索アイコンをクリックする
- **THEN** `"山田太郎" "ABCプロダクション"` をエンコードした URL が新しいタブで開く

### Requirement: 作品一覧テーブルの編集モード時にGoogle検索リンクを表示する
作品一覧テーブルにおいて、編集モードが有効なとき、各行の作品タイトルの横に Google 検索アイコンリンクを表示しなければならない（SHALL）。検索クエリは全出演者名・作品タイトル・`is_search_keyword = true` の作品用テキスト型カスタム項目の値（値が空でない場合のみ）をそれぞれダブルクォートで囲みスペースで結合した形式とする。リンクは新しいタブで開く。編集モードが無効のときは表示しない。

#### Scenario: 編集モード時に検索アイコンが表示される
- **WHEN** 作品一覧テーブルの編集モードが有効である
- **THEN** 各行の作品タイトルの横に Google 検索アイコンリンクが表示される

#### Scenario: 通常表示時に検索アイコンが表示されない
- **WHEN** 作品一覧テーブルの編集モードが無効である
- **THEN** 各行の作品タイトルの横に Google 検索アイコンリンクは表示されない

#### Scenario: カスタム項目値を含む検索クエリが生成される
- **WHEN** 出演者が「山田太郎」で作品タイトルが「ローマの休日」で is_search_keyword = true の text 型カスタム項目「品番」の値が「MIDE-123」のとき検索アイコンをクリックする
- **THEN** `"山田太郎" "ローマの休日" "MIDE-123"` をエンコードした URL が新しいタブで開く

#### Scenario: 出演者なしの場合の検索クエリが正しく生成される
- **WHEN** 出演者が登録されておらず作品タイトルが「ローマの休日」のとき検索アイコンをクリックする
- **THEN** クエリが `"ローマの休日"` をエンコードした URL が新しいタブで開く
