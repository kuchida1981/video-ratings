# Data Export / Import Spec

## Purpose

全登録データのバックアップ・環境移行手段を提供する。設定画面のデータ管理セクションから、エクスポート（ZIP形式、DBデータ＋カバー画像）とインポート（完全置換）を行える。

## Requirements

### Requirement: 全登録データをZIPファイルとしてエクスポートできる
設定画面のデータ管理セクションから、全テーブルのデータとカバー画像をZIPファイルとしてダウンロードできる。ZIPには `data.json`（DBデータ・スキーマバージョン・出力日時）と `covers/` ディレクトリ（全カバー画像）が含まれる。

#### Scenario: エクスポートの実行
- **WHEN** ユーザーが設定画面のデータ管理セクションで「エクスポート」ボタンをクリックする
- **THEN** ZIPファイル（`video-ratings-export-YYYY-MM-DD.zip`）がダウンロードされる

#### Scenario: ZIPにDBデータが含まれる
- **WHEN** エクスポートされたZIPファイルを展開する
- **THEN** `data.json` が存在し、全テーブルのデータと `schema_version`・`exported_at` が含まれる

#### Scenario: ZIPにカバー画像が含まれる
- **WHEN** エクスポートされたZIPファイルを展開する
- **THEN** `covers/works/` および `covers/performers/` にカバー画像ファイルが含まれる

#### Scenario: エクスポートファイルにスキーマバージョンが含まれる
- **WHEN** エクスポートされたZIPを展開し `data.json` を開く
- **THEN** トップレベルに `schema_version` フィールドが存在し、現在のAlembicリビジョン文字列が入っている

---

### Requirement: エクスポートしたZIPファイルから全データをインポートできる
設定画面のデータ管理セクションから、エクスポートと同形式のZIPファイルをアップロードして全データ（DBデータ＋カバー画像）を復元できる。インポートはDBデータ・カバー画像ともに完全置換（既存データを全削除してから再挿入・展開）で行われる。

#### Scenario: インポート前の確認ダイアログ表示
- **WHEN** ユーザーがZIPファイルを選択してインポートを実行しようとする
- **THEN** 「全データが置き換えられます」という警告とともに確認ダイアログが表示される

#### Scenario: 確認後にDBデータのインポートが完了する
- **WHEN** ユーザーが確認ダイアログでインポートを承認する
- **THEN** 既存の全DBデータが削除され、ZIPの `data.json` のデータが再挿入される

#### Scenario: 確認後にカバー画像のインポートが完了する
- **WHEN** ユーザーが確認ダイアログでインポートを承認する
- **THEN** 既存の `uploads/covers/` 配下が全削除され、ZIPの `covers/` の内容が `uploads/covers/` に展開される

#### Scenario: インポート成功後の通知
- **WHEN** インポートが正常に完了する
- **THEN** インポート完了のフィードバックが表示される

#### Scenario: schema_version 不一致時はインポートを拒否する
- **WHEN** アップロードされたZIPの `data.json` の `schema_version` が現在のDBリビジョンと異なる
- **THEN** インポートが実行されず、バージョン不一致のエラーメッセージが表示される

#### Scenario: 不正なZIPファイルはエラーになる
- **WHEN** ZIPでないファイルまたは `data.json` を含まないZIPファイルがアップロードされる
- **THEN** インポートが実行されず、エラーメッセージが表示される

---

### Requirement: 設定画面がカスタム項目とデータ管理の2セクションに整理される
設定ページ（`/settings`）は縦スクロール型のセクション構成になり、「カスタム項目」セクションと「データ管理」セクションを持つ。

#### Scenario: 設定画面の表示
- **WHEN** ユーザーが `/settings` に遷移する
- **THEN** カスタム項目セクションとデータ管理セクションが縦に並んで表示される
