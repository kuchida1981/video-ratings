# Capability: View Mode Persistence

## Purpose

作品一覧および出演者一覧の表示モード（タイル/リスト）をブラウザの localStorage に永続化し、ユーザーがページを再訪した際にも前回の表示設定が維持されるようにする。

## Requirements

### Requirement: 作品一覧の表示モードを永続化する
システムは作品一覧の表示モード（タイル/リスト）を `video-ratings:works-view-mode` キーで localStorage に保存し、ページ再訪時に復元しなければならない（SHALL）。保存されていない場合または不正な値の場合は `"tile"` をデフォルト値とする。

#### Scenario: 作品一覧でリスト表示に切り替える
- **WHEN** ユーザーが作品一覧でリスト表示ボタンをクリックする
- **THEN** 表示モードが `"table"` に切り替わり、localStorage の `video-ratings:works-view-mode` に `"table"` が保存される

#### Scenario: 作品一覧をページ再訪したとき前回のモードが復元される
- **WHEN** ユーザーが以前リスト表示を選択した後に作品一覧ページを再訪する
- **THEN** ページはリスト表示で表示される

#### Scenario: 作品一覧のフィルターをリセットしても表示モードは変わらない
- **WHEN** ユーザーがリスト表示を選択した状態でフィルターリセットボタンをクリックする
- **THEN** フィルター条件はリセットされるが、表示モードはリスト表示のまま維持される

### Requirement: 出演者一覧の表示モードを永続化する
システムは出演者一覧の表示モード（タイル/リスト）を `video-ratings:performers-view-mode` キーで localStorage に保存し、ページ再訪時に復元しなければならない（SHALL）。保存されていない場合または不正な値の場合は `"tile"` をデフォルト値とする。

#### Scenario: 出演者一覧でリスト表示に切り替える

#### Scenario: 出演者一覧をページ再訪したとき前回のモードが復元される
