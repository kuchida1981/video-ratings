# Spec: list-column-config

## Purpose

作品一覧・出演者一覧テーブルの表示列をユーザーが選択・カスタマイズできる機能の仕様。表示列の選択、カスタム項目・タグカテゴリの列追加、列設定の永続化、およびレイアウト維持に関するルールを定義する。
## Requirements

### Requirement: 列表示設定の廃止
作品一覧・出演者一覧はタイルグリッドとテーブルの両表示モードをサポートしなければならない（SHALL）。テーブル表示モードでは列表示選択のUIコントロールを提供しなければならない（SHALL）。タイルグリッド表示モードでは列選択UIを表示してはならない（SHALL NOT）。

#### Scenario: タイルグリッド表示時に列選択UIが表示されない
- **WHEN** ユーザーが作品一覧ページをタイルグリッド表示で開く
- **THEN** 列表示選択のUIコントロールは表示されない

#### Scenario: テーブル表示時に列選択UIが表示される
- **WHEN** ユーザーが作品一覧ページをテーブル表示に切り替える
- **THEN** フィルタパネル内に列表示選択のUIコントロールが表示される

#### Scenario: タイルグリッド表示時は作品がタイル形式で表示される
- **WHEN** ユーザーが作品一覧ページをタイルグリッド表示で開く
- **THEN** 作品はテーブル行ではなくタイルカードとして表示される
