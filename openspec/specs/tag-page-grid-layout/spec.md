# Spec: tag-page-grid-layout

## Purpose

タグ管理画面のカテゴリ一覧を2列グリッドのカードレイアウトで表示する。縦1列アコーディオンを廃し、画面横幅を活用することで情報密度を高め、視認性と操作性を向上させる。

## Requirements

### Requirement: タグ管理画面はカテゴリを2列グリッドで表示する
タグ管理画面のカテゴリ一覧は、縦1列リストではなく2列グリッドのカードレイアウトで表示されなければならない（SHALL）。各カテゴリは独立したカードとして表示され、「作品用カテゴリ」「出演者用カテゴリ」のセクション分けは維持される。

#### Scenario: カテゴリが2列に並ぶ
- **WHEN** タグ管理画面を表示する
- **THEN** 各セクション内のカテゴリカードが2列グリッドで配置される

#### Scenario: カードは自然な高さで表示される
- **WHEN** 同じグリッド行に異なるタグ数のカテゴリが並ぶ
- **THEN** 各カードはタグ数に応じた自然な高さで表示され、高さを無理に揃えない

### Requirement: タグリストは常時表示される
各カテゴリカード内のタグリストはアコーディオンで折りたたまず、常時表示されなければならない（SHALL）。

#### Scenario: タグリストが常に見える
- **WHEN** タグ管理画面を表示する
- **THEN** 全カテゴリのタグリストがすぐに確認できる（折りたたみ操作が不要）

### Requirement: グリッドレイアウトでカテゴリのDnD並び替えが機能する
カテゴリカードはグリッド内でドラッグ＆ドロップによる並び替えができなければならない（SHALL）。

#### Scenario: カテゴリをドラッグして並び替える
- **WHEN** カテゴリカードのグリップハンドルをドラッグする
- **THEN** グリッド内の別の位置にドロップでき、並び順が変更される

### Requirement: タグのDnD並び替えはカード内の縦リストで機能する
各カード内のタグは引き続き縦リストで表示され、ドラッグ＆ドロップによる並び替えが機能しなければならない（SHALL）。

#### Scenario: タグをドラッグして並び替える
- **WHEN** カード内のタグのグリップハンドルをドラッグする
- **THEN** 同じカード内で縦方向に並び替えができる
