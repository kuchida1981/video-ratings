# Tag Evaluation Spec

## Purpose

タグカテゴリ・タグの管理と、作品・出演者へのタグ付け、およびタグスコアに基づく合計スコア計算を担う機能を提供する。

## Requirements

### Requirement: タグカテゴリの管理
システムはタグカテゴリを作成・編集・削除できなければならない（SHALL）。カテゴリは `entity_type`（work または performer）と `is_multi_select`（複数選択可否）を持つ。作品用カテゴリと出演者用カテゴリは別プールとして管理する。

#### Scenario: 作品用カテゴリを作成できる
- **WHEN** ユーザーが entity_type=work でカテゴリを作成する
- **THEN** システムはそのカテゴリを保存し、作品評価時のみ表示される

#### Scenario: 出演者用カテゴリを作成できる
- **WHEN** ユーザーが entity_type=performer でカテゴリを作成する
- **THEN** システムはそのカテゴリを保存し、出演者評価時のみ表示される

#### Scenario: is_multi_select=false のカテゴリは1つのタグのみ選択可能
- **WHEN** ユーザーが is_multi_select=false のカテゴリから2つ目のタグを選択しようとする
- **THEN** システムは以前選択していたタグを外し、新しいタグのみを選択状態にする

#### Scenario: is_multi_select=true のカテゴリは複数タグを選択可能
- **WHEN** ユーザーが is_multi_select=true のカテゴリから複数のタグを選択する
- **THEN** システムは選択した全てのタグを保持する

---

### Requirement: タグの管理
システムはカテゴリに属するタグを作成・編集・削除できなければならない（SHALL）。タグはスコア（整数、nullable）を持つ。スコアがない場合はラベル専用タグとして機能する。タグの名前とスコアは作成後も変更できなければならない（SHALL）。

#### Scenario: スコアありのタグを作成できる
- **WHEN** ユーザーがスコアに整数値を入力してタグを作成する
- **THEN** システムはスコア付きのタグを保存する

#### Scenario: スコアなしのタグを作成できる
- **WHEN** ユーザーがスコアを入力せずにタグを作成する
- **THEN** システムはスコア null のタグを保存する（ラベル専用）

#### Scenario: タグ名を編集できる
- **WHEN** ユーザーが既存のタグの名前を変更して保存する
- **THEN** システムは更新された名前でタグを保存する

#### Scenario: タグのスコアを編集できる
- **WHEN** ユーザーが既存のタグのスコアを変更して保存する
- **THEN** システムは新しいスコアでタグを保存し、そのタグが付いた作品・出演者のスコアが再計算される

#### Scenario: タグのスコアを削除できる（スコアなしにできる）
- **WHEN** ユーザーが既存のタグのスコアを空にして保存する
- **THEN** システムはスコアを null に更新し、そのタグはスコア計算に影響しなくなる

#### Scenario: タグの編集をキャンセルできる
- **WHEN** ユーザーがタグ編集フォームでキャンセルする
- **THEN** システムはタグを変更せず、編集フォームを閉じる

#### Scenario: タグを削除すると既存のタグ付けも削除される
- **WHEN** ユーザーがタグを削除する
- **THEN** システムはそのタグを作品・出演者から外し、スコア計算に影響しなくなる

---

### Requirement: 作品へのタグ付け
システムは作品に対して作品用タグを付けられなければならない（SHALL）。カテゴリの is_multi_select ルールに従う。

#### Scenario: 作品にタグを付けられる
- **WHEN** ユーザーが作品詳細画面で作品用タグを選択する
- **THEN** システムはそのタグを作品に紐づけて保存する

#### Scenario: 作品からタグを外せる
- **WHEN** ユーザーが作品に付いたタグを外す
- **THEN** システムはタグの紐づけを削除し、スコアを再計算する

---

### Requirement: 出演者へのタグ付け
システムは出演者に対して出演者用タグを付けられなければならない（SHALL）。出演者評価はグローバルで、全作品のスコア計算に影響する。タグの付け外しは出演者詳細ページおよび作品詳細ページの両方から行える。

#### Scenario: 出演者にタグを付けられる
- **WHEN** ユーザーが出演者詳細画面または作品詳細画面で出演者用タグを選択する
- **THEN** システムはそのタグを出演者に紐づけ、その出演者が主演の全作品の合計スコアが更新される

#### Scenario: 出演者からタグを外せる
- **WHEN** ユーザーが出演者詳細画面または作品詳細画面で出演者に付いたタグを外す
- **THEN** システムはタグの紐づけを削除し、関連する作品の合計スコアが更新される

---

### Requirement: 作品の合計スコア計算
システムは作品の合計スコアを `Σ(作品タグのスコア) + 主演出演者の合計スコア` として計算・表示しなければならない（SHALL）。スコアなし（null）のタグは 0 として扱う。

#### Scenario: 作品タグと主演出演者タグの合計を表示する
- **WHEN** ユーザーが作品詳細画面を表示する
- **THEN** システムは作品タグのスコア合計と主演出演者タグのスコア合計を合算した値を表示する

#### Scenario: 主演出演者がいない場合は作品タグのスコアのみ
- **WHEN** 作品に主演出演者が設定されていない
- **THEN** システムは作品タグのスコア合計のみを合計スコアとして表示する

#### Scenario: スコアなしタグは合計スコアに影響しない
- **WHEN** 作品にスコア null のタグが付いている
- **THEN** そのタグは合計スコアの計算に含まれない（0 として扱われる）
